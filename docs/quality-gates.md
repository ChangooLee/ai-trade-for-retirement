# 품질 게이트 — 전략·백테스트 채택 기준

전략/신호가 화면의 "매수 후보"가 되려면 전부 통과해야 한다.

## G1. 룩어헤드 차단
- 신호 = 기준일 종가까지의 데이터, 집행 = 익일 시가 (오버나이트류는 종가 진입 명시·체결 가능성 평가)
- 주봉 신호 = 완성주만 (`calendar.last_completed_week_cutoff`)
- 유니버스 = 그 시점 기준으로 재구성(미래 거래대금/생존 정보 금지)

## G2. 비용
- 왕복 0.35% 기본 차감 + 0.5/0.7/1.0% 스트레스. 단타류는 중앙값이 비용을 넘는지 명시.

## G3. 통계 보정
- 날짜 군집: 종목 풀링 금지, 일별 동일가중 평균 = 1관측
- 기준선과 짝지은 차이(시장요인 상쇄) + t값 + 양일비율
- 보유기간 > 리밸런스 간격이면 비중첩 부분표본
- 집중도 진단(상위 1% 기여), 탈락률(거래정지) 진단

## G4. 생존편향
- **PIT 패널(pit_ohlcv) 검증 필수.** 생존 패널 결과는 참고로만 병기.
- PIT에서 상폐는 보수 처리(이후 40일 내 첫 호가, 없으면 전손)

## G5. 표기
- 화면 수치는 빌드 시 원데이터 재계산. 출처 스크립트를 validation-status.md에 기록.
- 부풀려진 수치 발견 시 즉시 철회 표기가 신규 기능보다 우선.

## G6. 실거래 관문 (현재 미충족 — 작업 금지 항목)
- 페이퍼 트레이딩 1개월+ 실적 일치 확인
- 월 손실 서킷브레이커 구현(예: 월 −3% 도달 시 당월 전량 현금)
- 사용자 명시 승인
