# 0006 — 집중도 상향: 시총 top300 → top400 (2026-06-13)

## 배경
사용자 요청: "백테스트(전체기간)에만 의존하면 올해 시장과 안 맞다. 거래대금이 올해 수익 좋으면 그게 낫지 않나?
리스크는 TDA로, 종목 선택은 공격적으로. 수익을 높이고 싶다." → 사용자 선택: **시총 유지 + 집중도↑(분할검증 통과 시에만 적용)**.

## 결정
- **유니버스 크기 top300 → top400.** top400은 양 반기 모두 +(견고), MDD↓(−43→−39), Sharpe↑(0.44→0.64),
  슬롯 5/7/10% 교차로 일관 → 단일 사이징/단일 N 우연이 아님.
- **포지션 집중(슬롯7~10%, 적게 보유)은 기각** — 수익을 못 높이고 top300에선 MDD를 −61%로 악화. 현행 슬롯5%/15종목 유지.

## 거래대금 가설 검증 (사용자 질문 직접 답)
거래대금 유니버스는 2026(+27 vs 시총 +30)·2024(+4 vs +3)만 비등/약간 우위. 다년 누적은 시총 압도
(최근4년 +72% vs +165%), MDD도 더 큼(−50 vs −39). → "공격적 거래대금이 수익을 높인다"는 데이터상 거짓.
시총 상위가 이 레짐의 주도주(AI·반도체·방산)를 담고 있어 공격성은 이미 시총 안에 있음.
