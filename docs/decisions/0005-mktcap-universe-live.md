# 0005 — 라이브 유니버스를 거래대금→시가총액으로 전환 (2026-06-13)

## 배경
결정 0003에서 시총 top300 유니버스가 PIT 검증을 통과(+67.7%, CAGR +5.9%)했으나, 라이브 신호 산출은
여전히 거래대금 상위 유니버스를 썼다(검증−운영 불일치). 거래대금 유니버스는 PIT −20.4%로, 풀 자체가
−49% 드리프트하는 투기 테마 풀이었다.

## 결정
라이브 유니버스를 **시가총액 상위 top_n (20일 평균 거래대금 ≥ min_trdval 종목 중)** 으로 전환.
- `config/strategy.yaml`: `top_n_by_trdval20`→`top_n`, `rank_by: mktcap`, `min_trdval: 5e8` 추가.
- `update_data.fetch_universe_trdval20`(라이브 일일 경로): MKTCAP 포착 + `avgval≥min_trdval` 필터 후 시총 정렬.
  → 백테스트(`avg_trdval20>5e8 → 시총 top300`)를 라이브에서 정확히 재현.
- `fetchers.fetch_top_liquid_universe`(전체 재수집 경로)·`krx_loader.build_daily_ohlcv`·`build_webapp`도 동일 전달.

## 부수 수정
- **`.env` 로더 버그**: `load_krx_auth_key`의 폴백 경로가 `app/.env`(한 단계 어긋남)라 로컬에서 키를 못 찾았다.
  서버는 cron이 환경변수를 export해 가려져 있었음. repo 루트까지 탐색하도록 `_load_env_key`로 일반화.

## 검증
- 시총 유니버스 dry-run: top12 = 삼성전자·SK하이닉스·SK스퀘어·삼성전기·현대차·LG엔솔… (실제 최대형주), 거래대금 유니버스와 86종목씩 상이(교집합 71%).
- 재구성 후 빌드: universe_count 297, 대형주 8종 전부 포함, 신호 정상 산출.
- PIT 재계산(시총 top300): +67.7%, CAGR +5.9%, MDD −43%, Sharpe 0.44, 승률 47%, 풀 +4.2% 대비 +63.5%p.
