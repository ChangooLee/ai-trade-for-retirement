# 전략 검증 현황 (정직 고지 — 갱신 의무)

> 규칙: 화면·문서의 모든 성과 주장은 이 파일에 출처(스크립트·패널)와 함께 기록한다.
> 패널: **생존**(daily_ohlcv — 현재 상장 상위, 부풀려짐) / **PIT**(pit_ohlcv — 상폐 포함, 신뢰 기준)

## 모멘텀·추세 (F리더 ∩ 20주선 눌림 + 40거래일 시간청산 + D4 노출)
- 생존 패널: +531% (2017–2026, CAGR 22.6%, Sharpe 1.29) — `backtest/portfolio_backtest.py`
- **PIT(거래대금 유니버스): −20.4%** (CAGR −2.5%) — 풀 자체가 −49% 드리프트(투기 풀)
- **PIT(시가총액 유니버스 top300): +67.7% (CAGR +5.9%, MDD −43%, 승률 47%)** — `backtest/pit_mktcap_backtest.py`
  - 풀 기준선 +4.2% 대비 **+63%p 초과 = 진짜 선별 알파**(생존편향 제거 후에도 양수)
  - 민감도: top200 +14%/top300 +68%/top500 +3.5% (top300 최적, 거래대금은 전 구간 음수 −19~−53%)
- **결론: 유니버스를 거래대금→시가총액으로 바꾸면 전략이 구제됨.** 단 CAGR +5.9%는 현실적 기대치(연 KOSPI +13% 동기간 대비 저조하나 생존편향 없는 진짜 수치). MDD −43%는 큼 → 리스크관리 필수.
  - 약점: 2018 −12·2019 −10·2022 −30·2024 −7(약세장 취약), 수익은 2023/2025/2026 강세장 집중 → 레짐 의존 여전

## TDA (Vietoris-Rips H1 지속성)
- 매수 알파 없음(생존 패널 교집합 검증, 기준선 이하) — `backtest/overlap_backtest.py`
- 가치: 매도/리스크 신호 — 청산 비교에서 보유 40→16일·변동성 28→19% (`backtest/tda_exit_backtest.py`)

## 오버나이트 1박 (거래량 3배+양봉 → 종가 매수, 익일 시가 매도)
- 건당: 평균 +0.53%/박, net(0.35%) +0.18%, 승률 54%, 중앙값 비용 미달 — `backtest/shortterm_signals_backtest.py`
- 포트폴리오(배포 룰): CAGR +1.3%/MDD −28% — `backtest/overnight_deployed_backtest.py`
- 국면 분해: **Risk-Off 밤만 net + (+0.17%/밤)**, Risk-On/Half는 엣지 없음 → 화면 경고가 이 부호를 따름
- 한계: 연도별 에피소드성(2020·2026 견인, 2022·2025 음). 노출 30% 이하 권장.
