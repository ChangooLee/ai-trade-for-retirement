> 패널: **생존**(daily_ohlcv — 현재 상장 상위, 부풀려짐) / **PIT**(pit_ohlcv — 상폐 포함, 신뢰 기준)

## 모멘텀·추세 (F리더 ∩ 20주선 눌림 + 40거래일 시간청산 + D4 노출)
- 생존 패널: +531% (2017–2026, CAGR 22.6%, Sharpe 1.29) — `backtest/portfolio_backtest.py`
- **PIT(거래대금 유니버스): −20.4%** (CAGR −2.5%, MDD −50.4%) — 풀 자체가 −49% 드리프트(투기 풀)
- **PIT(시가총액 유니버스 top400): +155.7% (CAGR +10.9%, MDD −39.2%, Sharpe 0.64, 승률 46%)** — `backtest/pit_mktcap_backtest.py --top 400`
  - 풀 기준선 −2.5% 대비 **+158p 초과 = 진짜 선별 알파**(생존편향 제거 후에도 강한 양수)
  - **집중도검증(`backtest/concentration_grid.py`, 결정 0006)**: top300(+68%, 전반 −8%=비견고) → **top400(+156%, 전반 +38%·후반 +85%=견고, MDD −39%·Sharpe 0.64)**. 슬롯 5/7/10% 교차로 견고(단일 사이징 우연 아님).
  - **N 민감성(과최적 주의)**: top350 +135%/top400 +156%/top450 +89%/top500 +3.5% — top400은 국소 정점. 견고 기대범위 **400~450 = +89~156%**. 포지션 집중(슬롯7~10%, 적게 보유)은 수익 미개선·MDD 악화로 **기각**(슬롯5%/15종목 유지).
- **결론: 유니버스를 거래대금→시가총액 top400으로. ✅ 라이브 적용(결정 0005·0006)** — `config rank_by=mktcap, top_n=400`, 화면 universe_rule="시가총액 상위 400".
  - 약점: 2018 −12·2019 −11·2022 −36(약세장 취약), 레짐 의존 여전(단 top400은 2024도 +3.3%로 개선).
  - **거래대금 vs 시총 연도별(올해 질문 검증)**: 거래대금은 2026(+27 vs +30)·2024(+4 vs +3)에서만 비등/약간 우위, **다년 누적은 시총 압도**(최근4년 거래대금 +72% vs 시총 +165%) AND 거래대금 MDD 더 큼(−50 vs −39). → 공격적 거래대금이 수익을 높이지 못함.
  - **정직 고지(가장 중요)**: 같은 기간 **KOSPI(시총가중) 매수후 보유 +257%(CAGR 15.1%)** > 전략 +156%(CAGR 10.9%, 격차 축소). 지수는 소수 초대형주가 견인 → '등가중 풀 대비 +158p'는 진짜 선별 알파지만 **인덱스 초과수익은 아님**(이 강세장 한정). 전략의 실제 가치는 **낮은 MDD·국면 분산·서킷브레이커 결합**에 있음.

## 전략 복합 (장기 모멘텀[시총] + 단타 오버나이트, 한 자본 — PIT)
`backtest/composite_strategy_backtest.py`. 일수익 상관 **+0.07**(거의 독립 — 분산효과 있음).
| 구성 | CAGR | MDD | Sharpe |
|---|---|---|---|
| 장기 100%(시총 top400·유동20억) | +8.0% | −32% | 0.51 |
| 단타 100%(오버나이트 Risk-Off) | −0.8% | −53% | 0.05 |
| 복합 80/20 | +6.7% | **−27%** | 0.50 |
| 복합 70/30 | +6.0% | −27.5% | 0.48 |
| 복합 50/50 | +4.3% | −29.5% | 0.38 |
- **결론**: 단타 추가는 **MDD를 −32%→−27%로 낮추나(상관 낮아 분산효과) 수익은 희석**(단타 net 엣지≈0).
  → 단타는 '수익원'이 아니라 소량(≤20%) '변동성 분산'으로만 의미. 주력은 장기 모멘텀(시총).
- 최선 근사: **장기 80% + 단타 20%** (CAGR +6.7%, MDD −27%, Sharpe 0.50) — 단 단타는 슬리피지 미반영 상한.
